Installation Guide

Step 1: Download the .zip file

A photo goes here

Head over to Disease Dynamics: Home1 and click on the Download Here button to download in your browser.

Step 2: Extract contents

A photo goes here

Navigate to your local Downloads folder and right-click to extract the zip. Within the extracted file, you will see following:

  • Backend: This folder contains all the files needed to run the program as well as the pre-loaded datasets. Within it, you will find the Datasets folder containing all of the pre-loaded datasets.
  • ML: This folder contains the trained models, one of which is currently used within Disease Dynamics.
  • Website: This folder contains the html and style sheet for the Disease Dynamics website.
  • DiseaseDynamics.exe: You can use this file to quickly start up the program.

Step 3: Run the program

To run Disease Dynamics on your device you can either:

  1. Run the DiseaseDynamics.exe file.

  2. Head over to the Backend folder and run the GUI.py file.

    Note: To run the python script, your device will need the following modules installed:

    PyQt5, matplotlib, geopandas, seaborn, pandas, numpy, joblib.

    If you do not have these installed, or are not sure, enter this command into your device Terminal:

    install PyQt5 matplotlib geopandas seaborn pandas numpy joblib

    To ensure the script runs as intended, take care not to move or delete any files within the Backend folder.

Step 4: Success!

You have successfully run Disease Dynamics on your device.
